# Fullstack Developer Bundle

A curated collection of commands for fullstack web development.

## Included Components

### Commands
- API design and implementation
- Frontend component creation
- Database design and queries
- Testing and debugging
- Git workflow

### Rules
- Fullstack project conventions

## Installation

```bash
# Copy commands
cp -r bundles/fullstack-developer/commands/* ~/.claude/commands/

# Copy project rules
cp bundles/fullstack-developer/CLAUDE.md ./CLAUDE.md
```

## Commands Included

| Command | Purpose |
|---------|---------|
| `/api-design` | Design API endpoints |
| `/api-endpoint` | Implement endpoints |
| `/api-test` | Test APIs |
| `/component` | Create UI components |
| `/style` | CSS and styling |
| `/a11y` | Accessibility audit |
| `/db-architect` | Database design |
| `/query` | Write queries |
| `/test` | Generate tests |
| `/debug` | Debug issues |
| `/commit` | Git commits |
| `/pr` | Create PRs |
| `/review` | Code review |

## Workflow

1. Design your API with `/api-design`
2. Create database schema with `/db-architect`
3. Implement endpoints with `/api-endpoint`
4. Build UI components with `/component`
5. Write tests with `/test` and `/api-test`
6. Review with `/review`
7. Commit with `/commit` and `/pr`
