# Web Development Plugin

Frontend and fullstack web development toolkit.

## Installation

```bash
claude plugin install ./plugins/web-development
```

## Included Commands

| Command | Description |
|---------|-------------|
| `/component` | Create UI components |
| `/style` | Help with CSS and styling |
| `/a11y` | Audit accessibility |
| `/api-endpoint` | Implement API endpoints |
| `/test` | Generate tests |

## Supported Frameworks

- React
- Vue
- Svelte
- Angular
- Next.js
- And more...

## Uninstall

```bash
claude plugin uninstall web-development
```
