# Frontend Commands

Commands for UI components, styling, and accessibility.

## Commands

| Command | Description |
|---------|-------------|
| `component` | Create UI components following project patterns |
| `style` | Help with CSS, styling, and design implementation |
| `a11y` | Audit and fix accessibility issues |

## Usage

```bash
# Create a component
/component user avatar with status indicator

# Fix styling
/style center this modal on the screen

# Audit accessibility
/a11y LoginForm component
```

## Installation

```bash
# Install all frontend commands
cp commands/frontend/*.md ~/.claude/commands/

# Install a single command
cp commands/frontend/component.md ~/.claude/commands/
```

## Supported Frameworks

- React (JSX, hooks)
- Vue (SFCs, composition API)
- Svelte
- Angular
- Solid
- Web Components

## Styling Approaches

- Plain CSS/SCSS/Less
- Tailwind CSS
- CSS Modules
- Styled-components/Emotion
- UI frameworks (MUI, Chakra, Radix, shadcn)
