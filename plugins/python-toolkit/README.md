# Python Toolkit Plugin

Python development essentials with database, testing, and documentation commands.

## Installation

```bash
claude plugin install ./plugins/python-toolkit
```

## Included Commands

| Command | Description |
|---------|-------------|
| `/db-architect` | Design and improve database schemas |
| `/db-migrate` | Generate database migration files |
| `/query` | Write and optimize database queries |
| `/test` | Generate tests for code |
| `/doc` | Generate or update documentation |

## Best For

- Django projects
- FastAPI projects
- Flask projects
- SQLAlchemy projects
- Any Python backend

## Uninstall

```bash
claude plugin uninstall python-toolkit
```
